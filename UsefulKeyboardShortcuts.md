# Useful Keyboard Shortcuts

## MacOS

* Change focused app
    * Cmd+Tab
* Switch between app windows
    * Cmd+`
* See all windows
    * Ctrl+Up
* Screenshot
    * Cmd+Shift+4
    * Cmd+Ctrl+Shift+4 to just copy the screen, not create a file.
* Screen recording
    * Cmd+Shift+5
* Quick open
    * Cmd+Space

## XCode

* Run build on target device
    * Cmd+R
* Stop process
    * Cmd+.
* Change target device
    * Ctrl+Shift+0 (#)
* Change build target
    * Ctrl+0 (#)
* Search all files
    * Cmd+Shift+F
* Open file
    * Cmd+Shift+O
* (Un)Comment highlighted section
    * Cmd+/
* Toggle breakpoint on selected line
    * Cmd+\
* Step over/into/out
    * F6/F7/F8
* Resume execution
    * Ctrl+Cmd+Y
* Jump to line number
    * Cmd+L
* Jump between code changes in a file
    * Ctrl+\
* Forward/back navigation
    * Ctrl+Cmd+Left/Right
* Indent selection
    * Ctrl+I
* Show current file in Project Navigator
    * Cmd+Shift+J

## Safari

* Close Tab
    * Cmd+W
* New Tab
    * Cmd+T
* Revive Last Closed Tab
    * Cmd+Shift+T
* Focus search bar
    * Cmd+L
* Back/Forward
    * Cmd+Left/Right
* Next/Previous tab
    * Ctrl+Tab or Ctrl+Shift+Tab

## Slack

* Next/Previous unread item
    * Option+Shift+Up/Down
* Back/Forward
    * Cmd+[ or Cmd+]
* Edit last sent message
    * Cmd+Up

## Radar

* Unscreen Radar (when sending to another team)
    * Cmd+K and type “Unscreen”, press enter
* New problem
    * Cmd+N
* Quick open
    * Cmd+Shift+O
* Copy link (with title)
    * Cmd+Shift+C
* Copy link (no title)
    * Ctrl+Cmd+Shift+C

## Mail

* Archive current item — easy way to keep a completely empty inbox
    * Cmd+Ctrl+A

## WebEx

* You can press Cmd+Shift+M to mute. There’s a setting to make this global, so unmute from anywhere.

## Custom shortcuts

Pro tip: You can set up your own shortcuts in Settings \ Keyboard \ Keyboard shortcuts
You can map to any menu item
* One I always set up is to map Zoom to Ctrl+Cmd+Z

* I found this useful for mapping shortcuts to Window → Move to {display}
    * ctrl+opt+cmd+left/right for a dual screen setup works great
